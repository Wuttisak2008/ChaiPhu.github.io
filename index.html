รับข้อเสนอ image(11).png ทำหน้านี้ยังไงไม่มีสีฟ้าอะ ภาพที่สร้างขึ้น: คำแนะนำการใช้งาน GitHub ในการพัฒนา แก้ไข ไม่หมายถึงสอนวิธไปหน้านี้ในgithub ถ้าหมายถึง หน้า Quick setup ใน GitHub ที่มีแถบสีฟ้า ตามรูป แต่ของคุณไม่มี แสดงว่า Repository ของคุณอาจไม่ใช่ Repository เปล่า (Empty Repository) หรือมีไฟล์อยู่แล้ว ให้ลองดังนี้ วิธีที่ 1 ถ้าเพิ่งสร้าง Repository ใหม่ เข้า Repository ของคุณ กดแท็บ Code ถ้ายังไม่มีไฟล์เลย จะขึ้นหน้า Quick setup เหมือนในรูป วิธีที่ 2 ถ้ามีไฟล์แล้ว (เช่น README.md) GitHub จะไม่แสดงกล่องสีฟ้า Quick setup แล้ว ให้กดปุ่ม Code (สีเขียว) แทน จะมี Clone HTTPS SSH GitHub CLI ซึ่งใช้แทนได้เหมือนกัน วิธีที่ 3 ถ้าต้องการให้ขึ้นเหมือนในรูป ต้องสร้าง Repository ใหม่โดย ไม่ติ๊ก Add a README file ไม่ติ๊ก .gitignore ไม่ติ๊ก License เมื่อสร้างเสร็จจะได้หน้า Quick setup พร้อมคำสั่ง git init git add . git commit -m "first commit" git branch -M main git remote add origin ... git push -u origin main ขอถามนิดครับ ตอนนี้ Repository ของคุณ มีไฟล์ README.md แล้วหรือยัง? หรือส่งรูปหน้า Repository ปัจจุบันมาได้เลย เดี๋ยวผมบอกว่าต้องกดตรงไหนให้ตรงกับของคุณครับ image(12).png มันบอกแบบนี้ จากรูปที่ส่งมา ขั้นตอนในเอกสารเป็นเวอร์ชัน GitHub เก่า ครับ ตอนนี้ GitHub เปลี่ยนหน้าตาแล้ว ทำให้หลายคนหาไม่เจอ ถ้าของคุณ ไม่มีแถบสีฟ้า Quick setup ให้ทำแบบนี้แทนได้เลย เข้า Repository ของคุณ กดปุ่ม Add file (มุมขวาบนเหนือรายการไฟล์) เลือก Create new file หรือถ้า Repository ยังไม่มีไฟล์เลย จะมีปุ่ม Create new file อยู่กลางหน้าแทน จากนั้นจะเข้าสู่หน้าที่เหมือน ข้อ 6 ในเอกสาร คือหน้าสำหรับตั้งชื่อไฟล์ (Name your file...) ถ้าของคุณหา Add file ไม่เจอ ส่ง ภาพหน้า Repository ของคุณเต็มหน้าจอ (ไม่ใช่รูปในเอกสาร) มาได้เลย เดี๋ยวผมชี้ตำแหน่งปุ่มให้ตรงกับหน้าจอของคุณ เพราะ GitHub มีหลายหน้าตาและขึ้นอยู่กับว่า Repository ของคุณมีไฟล์อยู่แล้วหรือยังครับ ขอโค้ดหนึ่งโค้ที่ทำให้ขึ้นหน้าเว็บได้ เกี่ยวกับพื้นที่โชว์ผลงาน ได้ครับ ด้านล่างเป็นโค้ด HTML ไฟล์เดียว สามารถสร้างเป็น index.html แล้วอัปโหลดขึ้น GitHub Pages ได้เลย เมื่อเปิดเว็บจะเป็นหน้า Portfolio (พื้นที่โชว์ผลงาน) แบบสวย เรียบง่าย และ Responsive Portfolio | ChaiPhu Channel
